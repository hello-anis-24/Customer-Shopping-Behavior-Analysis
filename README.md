# Customer-Shopping-Behavior-Analysis

This project analyzes customer shopping behavior using transactional data to understand how customers spend, what products they prefer, and how subscriptions, discounts, and reviews affect purchasing decisions.

The goal is to generate actionable business insights that help improve sales, customer engagement, and long-term growth.

# Objectives

Understand customer spending patterns

Identify high-value customer segments

Analyze the impact of discounts and subscriptions

Discover top-performing products and categories

Support data-driven marketing and business strategies

# Dataset Information

Total Records: 3,900 purchase transactions

Total Columns: 18

Key Data Includes:

Customer demographics (Age, Gender, Location, Subscription Status)

Purchase details (Product, Category, Price, Season, Size, Color)

Shopping behavior (Discounts, Promo Codes, Reviews, Shipping Type, Purchase Frequency)

# Missing Data:

37 missing values in the Review Rating column

# Tools & Technologies Used

Python – Data cleaning, preprocessing, and exploratory analysis

PostgreSQL – SQL queries for business insights and customer segmentation

Power BI – Interactive dashboards and data visualization

GitHub – Version control and project sharing

# Data Analysis Workflow
Data Preparation (Python)

Loaded and explored raw data

Handled missing review ratings using median values

Standardized column names

Created derived features (age groups, purchase frequency)

Prepared clean data for database storage

# SQL Analysis (PostgreSQL)

Revenue comparison by gender and age group

Subscriber vs. non-subscriber spending analysis

Identification of high-spending discount users

Customer segmentation (New, Returning, Loyal)

Top products by category and rating

Shipping type impact on purchase value

# Interactive dashboard for:

Revenue trends

Age group performance

Subscription impact

Discount effectiveness

Product and category insights

Designed for easy understanding by managers and stakeholders

# Key Insights

Adult customers contribute the highest revenue

Subscription customers spend more over time

Discounts increase purchase frequency but can reduce margins

High product ratings lead to repeat purchases

Express shipping users show higher satisfaction

Senior customers respond positively to discounts

# Business Recommendations

Promote subscriptions with exclusive benefits

Reward loyal customers to increase retention

Use discounts strategically to protect profit margins

Highlight best-selling and top-rated products

Focus marketing on high-revenue age groups

Improve online shopping experience and fast shipping options
